# FlowDesk

This application is for portfolio creation only. It may or may not be used in the future as a base for other projects.

It's an Intern Management System where users can create requests, tasks, and projects, with the possibility for management to supervise and help with development. It helps management prioritize tasks by assessing their importance and assigning them a higher priority if necessary.

## Main public

- Small companies
- Marketing agencies
- 50 people Startups

## Features

- Auth 
- Analytics Dashboard
- CRUD Demands
- Automated Priority System 
- Requests Kanban
- Search & Filters
- Comments
- Users Perfil
